# Flask Pet Store API

Backend REST API built with Python and Flask.

## Description
This project is a backend system for a pet store web application.  
It includes user authentication, product management, and database integration.

## Tech Stack
- Python
- Flask
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- Redis (basic caching)

## Features
- User registration and login
- JWT-based authentication
- CRUD operations
- REST API endpoints
- Relational database design

## Purpose
This project was built as a real-world practice backend application, simulating a production-ready API for a small business.
